# 4.14内核(64位)
### 适用于所有树莓派3的系统（包括raspbian）

## 内核安装方法

#### 克隆项目
`git clone https://github.com/chainsx/firmware64-rpi`
#### 安装内核
`cd install && sh install.sh`
#### 重启
`reboot`

- [X] Wirless Lan (WLAN)
- [X] Bluetooth (BT)
- [X] brcm80211
- [X] GPIO

如果install.sh在你的系统上出现问题了的话，请[手动安装内核](https://github.com/chainsx/firmware64-rpi/wiki/手动安装内核)
